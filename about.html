The purpose of this web page has been to create a resource for the players of Zelda Breath of the Wild.
The website is divided into two parts:
In the Home part the user will find the favorites part, where they will have their preferences saved and an interactive map of the game taken from the web page: Breath of the Wild Interactiv Map.
The Search part allows the user to search the main elements of the game to find relevant information about each one. You must first filter by category and all the elements of the category will appear, then you can filter by name, location or description. In this part, you can also "like" any object so that it is saved on the main page (Home) so that you can consult the information when you need it.
The project has been developed with the knowledge acquired during the Bootcamp and searching for internet resources. The API used has been found at Hyrule Compendium API.
